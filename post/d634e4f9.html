ResultMap中的属性

  • property：映射的 Java 类的属性名称。
  • column：数据库表中的列名。
  • javaType：用于指定映射的 Java 类型。
  • typeHandler：用于指定类型处理器，处理数据库列和 Java 属性之间的转换。
  • ofType：用于集合映射时，指定集合中元素的类型。
  • association：用于表示一对多或多对一关系映射。
